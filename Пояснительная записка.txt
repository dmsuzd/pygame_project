Пояснительная записка к проекту "Крестики-нолики"

1. Цель проекта
Создание интерактивной игры в крестики-нолики с системой учета статистики игроков, настройками звука и интерфейсом на основе Pygame.

2. Основной функционал
Главное меню с опциями: Играть, Настройки, Рекорды, Выход.
Ввод имен игроков перед началом игры.
Игровой процесс с анимированным отображением ходов (X/O).
Проверка победителя (по линиям и диагоналям) и ничьей.
Система рекордов с хранением статистики (победы, ничьи, поражения) в SQLite.
Настройки музыки (включение/выключение фоновой мелодии).

3. Используемые технологии
Python 3 + Pygame для графики, анимации и управления звуком.
SQLite для хранения статистики игроков.
ООП-подход (функции для управления экранами и логикой игры).

4. Особенности реализации
Анимация символов: плавное увеличение размера X/O при размещении на поле.
Сортировка рекордов: по убыванию побед → ничьих → возрастанию поражений.
Динамическое обновление БД: автоматическая запись результатов после каждой игры.
Адаптивный интерфейс: кнопки с обработкой событий hover и click.

5. Инструкция
Запустить main.py (предварительно установив зависимости: pygame, sqlite3).
Выбрать "Играть" → ввести имена игроков.
Кликать по клеткам для совершения ходов.
После завершения игры:
Перезапустить ("Перезапуск")
Вернуться в меню ("Назад").
В "Настройках" управлять музыкой, в "Рекордах" просматривать статистику.

6. Структура данных (SQLite)
Таблица records:
name (TEXT) — имя игрока (уникальное).
wins (INT) — количество побед.
draws (INT) — количество ничьих.
losses (INT) — количество поражений.
